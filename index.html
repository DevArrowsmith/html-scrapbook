Section 1: Block Elements

  • Start on a new line
  • End with a new line
  • Take up as much width as they can

Header

Headers are primarily used for titles.

  • <h1> is used for main titles, such as a page or website title. Only one should be used per page or view.
  • <h2> is used for secondary titles, eg. the section titles in this document, or titles for sections of an article.
  • <h3> onward are used for lower level titles.
  • It is best practice not to skip title levels, in part because this can interfere with screen reader use.
  • Although header elements adjust text size they should not be used for this purpose (unless it is a fortunate side effect of using a header appropriately as a title following the above rules). Instead use styling to modify text size.
  • You can use styling to adjust the size of header text if the default font size is inappropriate.

<h1>Header 1<\h1>
<h2>Header 2<\h2>
<h3>Header 3<\h3>
<h4>Header 4<\h4>
<h5>Header 5<\h5>
<h6>Header 6<\h6>

Ordered List

An ordered list is a numbered list. It consists of a series of <li> (list) elements wrapped in a <ol> (ordered list) element.

<ol>
<li>first li element<\li>
<li>second li element<\li>
<li>third li element<\li>
<\ol>

⬇️

  1. first li element
  2. second li element
  3. third li element

Unordered List

An unordered list is a list without enumeration. It consists of a series of <li> (list) elements wrapped in a <ul> (unordered list) element.

<ul>
<li>first li element<\li>
<li>second li element<\li>
<li>third li element<\li>
<\ul>

⬇️

  • first li element
  • second li element
  • third li element

Div

Div is an inline element used primarily as a container. It can be used to group elements, for example for styling or layout. It should only be used when no other semantic element better suits the intended purpose.

Span is similar to div except that span is an inline element.

<div><p>This is a paragraph element in a div element.<\p><\div>

This is a paragraph element in a div element.

Section 2: Inline Elements

  • Stay on the same line as their siblings
  • Take up only as much room as they need

Span

Span is an inline element used primarily as a container. It can be used to group elements, for example for styling or layout. It should only be used when no other semantic element better suits the intended purpose.

Div is similar to span except that div is a block element.

<span><p>This is a paragraph element in a span element.<\p><\span>

⬇️

This is a paragraph element in a span element.

Inline Quotation

Inline quotations are used for short quotations that don't require a line break. Browsers often implement Inline Quotation elements as text with added quotation marks, sometimes also italicised or with additional styles.

The Blockquote element should be used for longer quotations.

<q>In the beginning there was nothing, which exploded.<\q>

⬇️

In the beginning there was nothing, which exploded.

Inline Code

Inline code elements are used for text that is meant to represent computer code. Browsers implement inline code elements by styling them in a manner indicative of code, for example by using a distinctive monospace font.

<code>hello i am a robot bzzt 🤖<\code>

⬇️

hello i am a robot bzzt 🤖

Line Break

Line breaks produce a line break (carriage-return). Useful for poems!

<p>There was a robot from Manchester 🤖🐝<br>Funded by a stingy investor 💸<br>Its parts were outmoded 💾<br>And poorly coded 👨‍💻🍺<br>So now the robot is a jester 🤡<p>
